h1. batchzip

Short shell script, which zips files with same basename (ignoring the extension).

It can zip multi project files with the same name as f.i. project.txt, project.docx, project.psd, project.xfc, project.png, project.pdf in a directory to a sinlge zip file.

h2. Help

For available options and help use -h flag.
